# todoList-server
RestAPI와 JWT 이용한 TodoList 프로젝트

<br>

## 사용기술
Java, Spring Boot, Spring Security, Spring Data JPA, JWT, JPA, H2

<br>

## 구현기능
- 회원가입
- JWT 이용한 로그인 인증
- 할일 등록, 수정, 삭제 
- 할일 활성화/비활성화 처리

<br>
